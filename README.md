# Café De Kloeg — Website

Een karaktervolle Next.js website voor Café De Kloeg in Kralingen, Rotterdam.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push to GitHub
2. Import at [vercel.com](https://vercel.com)
3. Deploy

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
